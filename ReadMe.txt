数据需要经过局域网WIFI传到手机上。所以需要连上同一个WIFI，最好是5G频道。

第一步，安装电脑客户端，win7与win8是同一个安装包，win10是单独的安装包,4选1，安装一个即可。

第二步，安装 apk ， 电脑端0.9.15版本对应的apk是 0.9.41


第三步，在apk界面 可以看到已经检测到了电脑主机。


如果遇到问题，检查以下几个方面：
1.再次检查  是否连上了同一个  WIFI
2.再次检查  防火墙策略是否允许spacedesk通信，或者先临时允许所有网络通讯。
3.再次检查  PC端投屏功能是否为 ON状态
4.再次检查  是否安装包都是我提供的,否则会提示版本不匹配。
